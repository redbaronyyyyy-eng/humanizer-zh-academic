# Changelog

All notable changes to this project will be documented in this file.

## [2.1.0] - 2025-02-24

### Added

- **噪声预算概念** — 每千字允许保留 2-3 处轻微 AI 特征，防止过度修改
- **抗检测性评分维度** — 质量评分从 5 维扩展到 6 维（总分 /60）
- **适用范围说明** — 明确适用于中文学术写作，也适用于英文论文中文翻译稿
- **常见误区 #7** — 过度清洁反而产生新的不自然
- SOP 检查清单关联模式编号，便于查找

### Changed

- **模式合并** — 原模式10（通用积极结论）与模式17（泛化"重要意义"结尾）合并为新模式10，消除重复
- **模式编号连续化** — 去掉"新增"标签，所有模式统一编号 1-16
- **YAML frontmatter** — 增加 `activation`、`metadata` 字段，提高可发现性
- 质量评分总分从 /50 调整为 /60

### Fixed

- 模式示例差异化，避免改写前后示例重复

## [2.0.0] - 2025-02-04

### Added

- 基于真实论文实验（AIGC 率 >50% → 11%）的 17 种 AI 模式识别
- 强制硬约束表（7 项）
- 文体标定系统
- 四步 SOP 操作流程
- 完整改写示例
- 5 维质量评分体系
- 6 条常见误区

### Changed

- 从通用 AI 文本去味重构为中文学术写作专用
- 所有示例替换为中文学术论文场景

## [1.0.0] - 2024-12-01

- 初始版本，翻译自英文 humanizer skill
